Quick Start :

Clone it using git

$ git clone https://github.com/amine1107/Minimalize

Install with npm

$ npm install minimalize

Install with bower

$ bower install minimalize

Getting Started :

After downloading the framework, you will find 4 files.

  • Css
  • Scss : which contains the scss version of the framework
  • Js
  • Fonts

Usage :

After all is well, you may mention minimalize using any method.

<link rel"stylesheet" type="text/css" href="css/minimalize.css"></link>

Button :

Minimalize Offers you a vast choice among the buttons

The integration of the button is very simple, the class "btn" is by default. You can see the example above

Default button

Code :


      <!-- Default button -->

          <button class="btn">Default Button</button>

You can add class types for example if you want the button to be primary you add "btn primary"

Code :


      <!-- button with class -->

          <button class="btn primary">primary</button>
          <button class="btn success">success</button>
          <button class="btn warning">warning</button>
          <button class="btn disabled">disabled</button>

More Button Exemple

Adding classes to minimize is very simple


      <!-- button with class -->

          <button class="btn">Default Button</button>
          <button class="btn radius-8">Button with radius</button>
          <button class="btn radius-8 emerald">Button with radius color</button>
          <button class="btn radius-8 hover asphalt">Button with radius color and hover</button>

Button Radius :

Code :


      <!-- button radius -->

          <button class="btn radius-6 ">Button</button>
          <button class="btn radius-8">Button</button>
          <button class="btn radius-10">Button</button>

Largest and smallest button :

Code :


      <!-- button  -->

Navbar

Minimalize menu is very simple, the framework offers you a classic and responsive menu

brand
link1 link2 link3

Code :


                <!-- navbar  -->

          <div class="navbar-component">

                 <!-- class 'area' is a container  -->

              <div class="navbar area">

                  <!-- logo  -->

                  <a href="#" class="brand">brand</a>

                  <!-- list of links  -->

                  <nav role="navigation" id="navigation" class="list">
                      <a href="#" class="item -link">link1</a>
                      <a href="#" class="item -link">link2</a>
                      <a href="#" class="item -link">link3</a>
                  </nav>

                  <!-- Button to toggle the display menu  -->

                  <button data-collapse data-target="#navigation" class="toggle">

                  <!-- Hamburger icon  -->

                      <span class="icon"></span>
                  </button>
              </div>
          </div>
